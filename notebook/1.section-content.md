# XCSS Documentation

## Intuitive Structure

_Style Like You Think_

XCSS’s human-readable syntax (`$normals { @assemble d-flex; }`) feels like CSS but scales effortlessly. Metaclasses (`L14__*`) link to source files (e.g., `app.html.xcss:14`), speeding up debugging (~30-60s/style). With a 15-minute learning curve and 4/10 effort, it’s ideal for SPAs, non-JS frameworks, or raw HTML—style templating languages like Pug or Handlebars with ease, outpacing Sass’s complexity and Tailwind’s clutter. **When CSS is predictable**, XCSS ensures styles behave consistently, eliminating surprises from cascading or specificity conflicts.

**Features:**

- Human-readable syntax with scalability
- Metaclasses for fast debugging and source mapping
- Quick learning curve (15 minutes) and low effort (4/10)
- Ideal for SPAs, non-JS frameworks, and raw HTML
- Compatible with templating languages (e.g., Pug, Handlebars)
- Predictable CSS behavior for reliable styling

## Dynamic Libraries

_Grow Without the Groan_

XCSS’s Axiom (253 styles) and Library (174 styles) adapt as you code. Add `new-component.xcss`, and styles like `.f-bold` auto-integrate via `@assemble`, keeping bundles at 10-19KB (~40-60% smaller than Tailwind). No manual imports—style raw HTML or templated pages (e.g., Jinja2) with zero overhead, leaving Sass’s bloat behind.

**Features:**

- Adaptive Axiom and Library indices
- Automatic style integration via `@assemble`
- Lean bundles (10-19KB), ~40-60% smaller than Tailwind
- No manual imports required
- Zero overhead for raw HTML or templated pages

## Dynamic Documentation

_Debug Like a Detective_

XCSS’s manifests (`{ "metaclass": "L14__*", "source": "$normals" }`) and **self-commented origin for each property** (e.g., `/* xtyles :: AXIOM => :root */`) trace styles to their roots, cutting debugging to ~30-60s/style. The **assistive VSCode extension** provides instant insights and suggestions, while **latest docs from terminal** offer quick reference. Perfect for non-JS frameworks or raw HTML, it boosts team productivity (~5-10 hours/week for 10 devs).

**Features:**
