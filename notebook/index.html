Intuitive & Predictable Styling

Style Like You Think

Stop fighting your CSS. XCSS's human-readable syntax ($normals { @assemble d-flex; }) feels like natural CSS but scales effortlessly. Say goodbye to specificity nightmares—XCSS ensures styles behave consistently, eliminating surprises from cascading or specificity conflicts.

  • Human-Readable Syntax: Write CSS that makes sense.
  • Predictable Behavior: Consistent styling, every time.
  • Rapid Learning Curve: Master XCSS in just 15 minutes (4/10 effort).
  • Ideal for Any Project: SPAs, non-JS frameworks, raw HTML, and templating languages (Pug, Handlebars, Jinja2). 

/* XCSS Example */
$button {
    @assemble d-flex items-center justify-center;
    padding: 1rem 2rem;
    border-radius: 9999px;
    background-color: #6d28d9; /* purple-700 */
    color: white;
    font-weight: 700;
    transition: all 0.3s ease;
}

$button:hover {
    background-color: #5b21b6; /* purple-800 */
    transform: scale(1.05);
}

Effortless Debugging & Maintenance

Grow Without the Groan: Zero Tech Debt, Maximum Productivity

XCSS automates the tedious, so you can focus on innovation. Debug styles in seconds, integrate new components seamlessly, and eliminate styling tech debt.

  • Dynamic Libraries: Axiom (253 styles) and Library (174 styles) adapt and integrate new styles automatically via @assemble—no manual imports.
  • Dynamic Documentation: Debug like a detective with self-commented origins (/* xtyles :: AXIOM => :root */) and metaclasses (L14__*) that trace styles to their source (e.g., app.html.xcss:14). Cut debugging time to 30-60s/style.
  • Assistive VSCode Extension: Instant insights and suggestions right in your editor.
  • "CSS Commits For You": XCSS automates style updates, drastically reducing manual effort and version control hassles.
  • Near Zero Tech Debt: Minimize styling overhead and keep your codebase clean as you scale.
